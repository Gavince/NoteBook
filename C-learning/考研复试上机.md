### (12)用正方形覆盖矩形

>```markdown
>1，问题描述
>有一矩形和以正方形。输入矩形的长n，宽m，和正方形边长a。输出至少多少个这样正方形能把该矩形完全覆盖（*当然覆盖面积大于矩形*）
>2，问题思路
>该该问题主要在于分类讨论，有三种情况①当a > n时，此时最简单，一个正方形便可完全覆盖长方形。②当n > a >m时，此时只要求出 n/a+1,那便是需要的正方形数目。③当 a < m时，此时个数为（n/a+1）*(m/a+1)。
>```

```c
# include <stdio.h>

int Cal_area(int, int , int);
int main()
{
    int a, b, n;
    int sum;

    printf("Please enter a length(a):");
    scanf("%d", &a);
    printf("Please enter a width(b):");
    scanf("%d", &b);
    printf("please enter a n:");
    scanf("%d", &n);
    sum = Cal_area(a, b, n);
    printf("小正方形的个数:%d\n", sum);

    return 0;
}

int Cal_area(int a, int b, int n)
{
    int sum;

    if (n < b) //如果边长比宽度小
    {
        sum = (a/n+1)*(b/n+1);
    }
    else if (n < a) //如果边长比长度小
    {
        sum = a/n+1;
    }
    else //刚刚好
    {
        sum = 1;
    }

   return sum;
}
```
